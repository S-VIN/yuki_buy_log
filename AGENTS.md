В папке postgres находится:
- shema.sql - sql скрипт для postgres, который можно запустить, чтобы пересоздать БД.
- "postgres://postgres:postgres@localhost:5432/yuki_buy_log?sslmode=disable" - строка подключения к БД
- testdata.sql - скрипт, создающий в БД тестовые данные для ручного тестирования. Этот скрипт без явных инструкций на БД не надо накатывать

В папке server находится код сервера на golang. 
- сервер работает через http api на порту 8080
- сервер логгирует свою работу в консоль
